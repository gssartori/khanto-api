Bem-vindo ao Projeto Django

Este projeto é uma demonstração de APIs REST usando Django e Django Rest Framework.

Utilize a descrição para utilização da API abaixo:

API de Imóveis:

  • Listar Imóveis: /imoveis/ - Obtém uma lista de todos os imóveis disponíveis.
  • Detalhes do Imóvel: /imoveis/<int:pk>/ - Obtém os detalhes de um imóvel específico com base no ID.
  • Criar Imóvel: /imoveis/create/ - Cria um novo imóvel.
  • Atualizar Imóvel: /imoveis/update/<int:pk>/ - Atualiza as informações de um imóvel existente com base no ID.
  • Excluir Imóvel: /imoveis/destroy/<int:pk>/ - Exclui um imóvel existente com base no ID.

API de Anúncios:

  • Listar Anúncios: /anuncios/ - Obtém uma lista de todos os anúncios disponíveis.
  • Criar Anúncio: /anuncios/create/ - Cria um novo anúncio.
  • Detalhes do Anúncio: /anuncios/<int:pk>/ - Obtém os detalhes de um anúncio específico com base no ID.

API de Reservas:

  • Listar Reservas: /reservas/ - Obtém uma lista de todas as reservas feitas.
  • Criar Reserva: /reservas/create/ - Cria uma nova reserva.
  • Detalhes da Reserva: /reservas/<int:pk>/ - Obtém os detalhes de uma reserva específica com base no ID.
